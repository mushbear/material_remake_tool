# 20260318视频剪辑脚本

## 📋 概述

自动化视频剪辑工具，根据叙事功能标签自动剪辑和合并视频片段。

## 🎯 剪辑规则

### 1. 场景使用规则
- **每个素材的每个场景只能使用一次**
- **F06-悬念结尾/付费卡点可以跨方案复用**
- **同一方案避免重复使用同一素材**（尽可能分散使用）

### 2. 场景合并规则
- **连续相同打标的片段自动合并**
- 合并后的场景被视为一个场景，只能使用一次
- 示例：F02片段(23-200s) + F02片段(200-262s) → 合并为F02(23-262s)

### 3. 时长限制规则
- **F01-强开局/吸睛钩子**：不限制时长
- **F06-悬念结尾/付费卡点**：不限制时长
- **其他所有场景**：必须>15秒

### 4. 最后场景处理规则
- **如果是素材的最后片段且不是F06**：去掉最后1秒
- 示例：F04片段(550-998s)是最后片段 → 输出(550-997s)

## 🎬 支持的叙事标签

| 标签 | 说明 | 时长限制 |
|------|------|----------|
| F01-强开局/吸睛钩子 | 开场吸引注意力 | 无限制 |
| F02-背景速递/设定交代 | 背景故事交代 | >15秒 |
| F03-极限施压/受辱 | 冲突压力场景 | >15秒 |
| F04-金手指觉醒/身份曝光 | 身份揭露/反转 | >15秒 |
| F05-高潮打脸/绝地反击 | 高潮反击 | >15秒 |
| F06-悬念结尾/付费卡点 | 悬念结尾/付费引导 | 无限制 |
| F07-情感拉扯/发糖 | 情感戏/发糖 | >15秒 |
